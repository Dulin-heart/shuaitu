# shuaitu
率土荣誉值计算器
🏆 赛季荣誉值精准计算器 (Honor Calculator)本项目是一款专门为战略手游《率土之滨》玩家定制的赛季结算荣誉值模拟工具。
通过简单的网页交互，玩家可以快速预估自己在赛季末能获得的荣誉点数，从而合理规划游戏进度。
✨ 核心功能全维度覆盖：整合了“攻城篇”、“势力篇”、“武勋篇”、“打架篇”四大维度的计算逻辑。
实时动态计算：无需点击提交按钮，输入数值的同时，底部汇总面板会实时更新各项荣誉与总计。
自动阶梯算法：内置复杂的阶梯式计算逻辑（例如拆迁值超过10万后的收益衰减）。上
限自动校准：严格遵循游戏规则中的各项上限设定，防止过度预估。
移动端适配：响应式设计，支持在手机浏览器、微信、钉钉等移动端完美显示。
📊 计算逻辑如下表所示
攻城荣誉:杀敌 1万=1荣誉；拆迁 10万内按1000=1，超10万按3000=1,杀敌上限100
势力值：3万以下 1万≈70荣誉；3万以上 1万≈35荣誉,总上限300
武勋值：200万以下 1万≈1荣誉；200万以上 2.5万≈1荣誉,无明确上限
征战表现：翻地每10块=1；打架拆迁每1000=1,翻地100 / 拆迁200
